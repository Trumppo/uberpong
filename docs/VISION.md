# Vision Summary

Tämä repository sisältää toteutuksen liitteenä olleen "UBERPONG: Pinball Court" -vision pohjalta.

Vision ydinkohdat, jotka on toteutettu:

- Pong-ydinmekaniikka säilytetty
- Tempojärjestelmä (rallin aikana kasvava intensiteetti)
- Kombolaskuri (onnistuneet torjunnat)
- Riskialueet (staattiset bonusalueet)
- Slap Shot -erikoiskyky cooldownilla
- Pelimuodot: Casual, Arena, Endurance
- Staattinen verkkototeutus ilman backendiä
- Konfiguroitavuus JSON-tiedostolla
- Paikallinen ennätys localStorageen
